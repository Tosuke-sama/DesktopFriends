# 简介

DesktopFriends 是一个开源的 Live2D 桌面宠物应用，让你可以将闲置的旧手机变成可爱的桌面伙伴。

## 主要特性

### 🎭 Live2D 支持
- 支持加载自定义 Live2D 模型
- 流畅的动画和表情切换
- 触摸交互响应

### 💬 AI 对话
- 支持多种 AI 后端：
  - Ollama（本地运行）
  - OpenAI
  - Google Gemini
- 自然语言对话
- FunctionCall 让人物拥有动态的表情动作
- 语音输入和 TTS 输出（开发中）

### 📱 多设备联动
- 手机端与电脑端均可开启房间（中继服务器）
- 局域网自动发现，搜索房间（mDNS）
- 信息实时同步并共享

## 工作原理

```
┌─────────────────┐     Socket.IO      ┌───────────────────────┐
│   中继服务器      │ ◄───────────────► │   应用端 (手机或者电脑)   │
│  Relay Server   │      WebSocket     │   Live2D App          │
└─────────────────┘                    └───────────────────────┘
        │
        │ HTTP API
        ▼
┌─────────────────┐
│   AI 服务       │
│ Ollama/OpenAI   │
└─────────────────┘
```

1. **应用端**：安卓手机运行 Live2D 应用，展示角色和动画
2. **中继服务器端**：运行中继服务器，处理 AI 对话和设备控制
3. **通信**：通过 Socket.IO 在局域网内实时通信
