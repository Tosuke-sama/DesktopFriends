# macOS 应用代理插件

让桌宠能够访问和控制 macOS 内置应用，如备忘录、日历等。

## 功能

### 备忘录（Notes）

- **读取备忘录**：获取所有备忘录列表和内容
- **创建备忘录**：创建新的备忘录（计划中）
- **编辑备忘录**：修改现有备忘录（计划中）
- **删除备忘录**：删除备忘录（计划中）

### 日历（Calendar）

- **读取事件**：获取日历事件（计划中）
- **创建事件**：创建新的日历事件（计划中）

## 使用方法

### 权限设置

1. 打开插件管理器
2. 点击「macOS 应用代理」插件
3. 在权限设置中启用需要的权限：
   - ✅ 备忘录读取
   - ✅ 备忘录创建（计划中）
   - ✅ 备忘录编辑（计划中）
   - ✅ 备忘录删除（计划中）

### 与桌宠交互

启用权限后，你可以直接询问桌宠：

- "读取我的备忘录"
- "我的备忘录里有什么？"
- "帮我看看备忘录里关于 XXX 的内容"

桌宠会自动调用插件工具来读取备忘录信息。

## 提供的 AI 工具

| 工具名 | 功能 | 所需权限 |
|--------|------|----------|
| `read_notes` | 读取所有备忘录列表 | 备忘录读取 |

## 注意事项

- ⚠️ **macOS 专用**：此插件仅在 macOS 上可用
- ⚠️ **权限要求**：首次使用需要授予应用相应的系统权限
- ⚠️ **隐私保护**：所有数据仅在本地处理，不会上传到任何服务器
- ⚠️ **性能考虑**：读取大量备忘录时可能需要一些时间

## 系统要求

- macOS 10.14 或更高版本
- 已安装备忘录应用（Notes.app）

## 故障排除

### 无法读取备忘录

1. 检查是否在插件设置中启用了「备忘录读取」权限
2. 确认 macOS 系统权限中已授予 TableFri 访问备忘录的权限
3. 尝试重启应用

### AppleScript 执行失败

如果遇到 AppleScript 执行错误，可能是：
- 备忘录应用未运行（插件会自动启动）
- 系统权限未授予
- macOS 版本不兼容
