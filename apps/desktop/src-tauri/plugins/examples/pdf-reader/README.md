# PDF 阅读器插件

让你的桌宠帮助你阅读和理解 PDF 文档。

## 功能

- **拖拽打开**：将 PDF 文件拖拽到桌宠窗口，自动打开阅读器
- **文本选择**：在 PDF 中选中文字，自动发送给桌宠处理
- **高清渲染**：支持 Retina 显示屏的高清渲染
- **页面导航**：支持滚动浏览和缩放控制

## 使用方法

### 打开 PDF 文件

1. 将 PDF 文件拖拽到桌宠窗口
2. 或在插件管理器中点击「打开」按钮

### 文字交互

1. 在 PDF 阅读器中选中文字（至少 3 个字符）
2. 等待 1 秒后，选中的文字会自动发送给桌宠
3. 桌宠会根据上下文帮助你：
   - 翻译外语内容
   - 解释专业术语
   - 总结段落要点
   - 回答相关问题

### 缩放控制

- 点击 `+` 放大页面
- 点击 `-` 缩小页面
- 缩放范围：50% - 300%

## 提供的 AI 工具

桌宠可以使用以下工具来帮助你处理 PDF：

| 工具 | 功能 |
|------|------|
| `open_pdf` | 打开 PDF 文件 |
| `read_pdf_page` | 读取指定页面内容 |
| `get_pdf_info` | 获取文档信息 |
| `search_pdf` | 在文档中搜索文本 |
| `get_selected_text` | 获取当前选中的文本 |

## 注意事项

- 部分扫描版 PDF 可能无法选中文字
- 大型 PDF 文件首次加载可能较慢
- 需要网络连接以加载 PDF.js 渲染引擎
